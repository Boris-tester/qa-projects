# Yandex Scooter Testing

Проект по проектированию тестов и ручному тестированию веб-приложения сервиса аренды самокатов **Яндекс Самокат**.

## Цель проекта

Подготовить тестовую документацию и проверить корректность работы ключевых пользовательских сценариев веб-приложения, а также спроектировать тест-кейсы для мобильного приложения.

## Что тестировалось

### Веб-приложение
В рамках проекта проверялись:

- экран «Статус заказа»
- экран «Сделать заказ»
- валидация полей формы
- отображение интерфейса по требованиям и макетам
- обработка ошибок
- работа приложения в разных браузерах и разрешениях экрана

### Мобильное приложение
Для мобильного приложения были спроектированы тест-кейсы на заданную функциональность, включая:

- пользовательские сценарии
- проверки интерфейса по макетам
- проверки вёрстки экранов

## Что было сделано

В рамках проекта были выполнены:

- анализ требований к веб- и мобильному приложению
- составление чек-листа для экрана «Статус заказа»
- подготовка проверок на валидацию полей формы
- проведение функционального тестирования веб-приложения
- оформление баг-репортов по найденным дефектам
- проектирование тест-кейсов для мобильного приложения
- проверка интерфейса и вёрстки по макетам Figma
- фиксация результатов выполнения тестов

## Используемые инструменты

- Google Sheets / Excel
- Google Chrome
- Yandex Browser
- Figma
- Chrome DevTools
