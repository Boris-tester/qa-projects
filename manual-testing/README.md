# Manual Testing Projects

В этом разделе представлены мои проекты по ручному тестированию веб- и мобильных приложений.

В рамках проектов были выполнены:

- функциональное тестирование
- регрессионное тестирование
- тестирование пользовательских сценариев
- составление тест-кейсов и чек-листов
- оформление баг-репортов
- ретест исправленных дефектов
- тестирование API
- проверка данных с помощью SQL

## Инструменты

- Google Sheets / Excel — ведение тестовой документации  
- Postman — тестирование API  
- Chrome DevTools — анализ сетевых запросов  
- Charles Proxy — анализ HTTP-трафика  
- Android Studio — тестирование и анализ работы мобильного приложения  
- Figma — проверка соответствия интерфейса макетам
