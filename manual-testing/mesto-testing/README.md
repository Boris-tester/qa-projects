# Mesto Web Application Testing

Проект по ручному тестированию веб-приложения Mesto.

## Что тестировалось

- функциональность веб-приложения
- пользовательские сценарии
- корректность работы интерфейса
- обработка ошибок

## Что было выполнено

- составлены баг-репорты
- проведено регрессионное тестирование
- выполнен ретест исправленных дефектов

## Инструменты

- Google Sheets / Excel
- Chrome DevTools
