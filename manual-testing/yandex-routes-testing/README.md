# Yandex Routes Testing

Проект по ручному тестированию веб-приложения **Яндекс Маршруты**.

## Что тестировалось

В рамках проекта проверялась функциональность сервиса построения маршрутов, включая:

- элементы интерфейса и вёрстку
- пользовательские сценарии построения маршрута
- корректность отображения данных
- поведение формы бронирования
- обработку ошибок
- работу приложения в разных браузерах и разрешениях экрана

## Что было сделано

В рамках проекта были выполнены:

- тест-анализ требований
- составление чек-листов
- подготовка проверок с использованием техник тест-дизайна
- написание тест-кейсов
- проведение функционального тестирования
- проверка вёрстки в разных окружениях
- оформление баг-репортов по найденным дефектам

## Инструменты

- Google Sheets / Excel
- Google Chrome
- Yandex Browser
- Firefox
- DevTools

## Файлы проекта

- yandex-routes-manual-testing.xlsx — тест-анализ, техники тест-дизайна, тест-кейсы и баг-репорты
- yandex-routes-testing-docs.xlsx — чек-листы проверки вёрстки и функциональности, тест-кейсы пользовательских сценариев и найденные дефекты
