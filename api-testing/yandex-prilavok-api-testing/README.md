# Yandex Prilavok API Testing

Проект по функциональному тестированию API сервиса доставки продуктов **Яндекс Прилавок**.

## Цель проекта
Проверить корректность работы API сервиса: обработку запросов, корректность ответов сервера и обработку ошибок.

## Что тестировалось

В рамках проекта проверялись следующие сценарии:

- добавление продукта в корзину
- повторное добавление продукта
- обработка невалидных параметров
- проверка отсутствующих обязательных параметров
- обработка несуществующих ID
- проверка граничных значений параметров
- проверка сообщений об ошибках
- проверка корректности статус-кодов API

## Что было сделано

- анализ API документации
- составление чек-листа для проверки API
- выполнение тестирования через API-запросы
- проверка структуры ответов сервера
- проверка статус-кодов
- оформление найденных дефектов

## Используемые инструменты

- Postman
- REST API
- JSON
- Excel / Google Sheets
