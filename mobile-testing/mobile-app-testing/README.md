# Mobile App Testing

Проект по ручному тестированию мобильного приложения.

## О проекте
В рамках проекта проводилось функциональное и регрессионное тестирование мобильного приложения для построения маршрутов.

Тестирование выполнялось на эмуляторе Android через **Android Studio**.

## Что тестировалось

- построение маршрутов
- выбор станций на карте
- отображение карточки станции
- история маршрутов и станций
- обновление маршрута по времени
- отображение деталей маршрута
- масштаб карты при смене ориентации
- сохранение состояния карточек и настроек при повороте экрана

## Что было сделано

В рамках проекта были выполнены:

- анализ требований приложения
- составление чек-листов
- проведение функционального тестирования
- проведение регрессионного тестирования
- оформление баг-репортов по найденным дефектам

## Инструменты

- Android Studio
- Android Emulator
- Google Sheets / Excel
