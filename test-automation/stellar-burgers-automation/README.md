# Stellar Burgers Automation

Проект по автоматизации тестирования веб-приложения Stellar Burgers.

## О проекте

В рамках проекта были реализованы три направления автоматизированного тестирования:

- unit-тесты
- API-тесты
- UI-тесты

## Стек

- Java
- JUnit 4
- Mockito
- JaCoCo
- REST Assured
- Selenium WebDriver
- Allure
- Maven

## Структура проекта

- `unit-tests` — unit-тесты бизнес-логики
- `api-tests` — API-тесты пользовательских и заказных сценариев
- `ui-tests` — UI-тесты веб-приложения Stellar Burgers

## Что проверяется

### Unit tests
- логика формирования бургера
- работа ингредиентов и булок
- покрытие кода unit-тестами

### API tests
- создание пользователя
- авторизация пользователя
- создание заказа
- негативные сценарии API

### UI tests
- регистрация пользователя
- вход в аккаунт
- переходы между разделами конструктора
- пользовательские сценарии интерфейса

## Запуск

Для запуска тестов используется Maven.

## Примечание

Проект разделён на три части для удобной навигации по видам автоматизированного тестирования.
